Propiedades del evento

El objeto event proporciona diversas propiedades que contienen informacion sobre el evento occurido. Alguna de las propiedades mas comunes incluye:


type

La propiedad type es ub objeto de evento indica el tipo de evento que ha occurido. Es una cadena de texto que identifica la naturalexza del evento. Por ejemplo, si estamos escuchando un clic del raton, la propiedad type contendra la cadena "click". De manera similar, ventos como "keydown" representaran pulsaciones de teclas, mientras que "mouseover" indicara que el raton se ha movido sobre un elemento. En resumen, la propiedad type proporciona informacion sobre la categoria especifica de evento que ha sido desencenado.

La propiedad 'timestamp' en un objeto de evento representa la marca de tiempo del momento en que occurio el evento. Esta marca de tiempo se mide en milisegundos desde el inicio del tiempo Unix (1 de enero de 1970, 00:00:00 UTC) La propiedad "timestamp" es util para determinar la secuencia temporal de ventos y puede ser utilizada para calcular la duracion entre eventos o para sincronizar acciones basadas en ele tiempo en una aplicacion.
